# WeatherApp

An MVC design pattern conform, opensource weather app for iOS. It gathers data from the OpenWeatherMap public API.

## Features
- [x] MVC design
- [x] 5 day forecast
- [x] Detailed current weather
- [x] Offline functionality
- [ ] Worldwide coverage
